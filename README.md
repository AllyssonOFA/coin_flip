## What is this?
- This is a TUI solution for the [Flip The Coin](https://devchallenges.io/challenge/flip-the-coin) challenge at [devchallenges.io](https://devchallenges.io/)

## Installation
> Make sure you have the latest version of pip installed.

### Method 1 - Building from source
```bash
# Clone this repository
git clone https://github.com/AllyssonOFA/coin_flip.git

# move to directory
cd coin_flip

# build the project
python -m pip install .

# or
pip install .

# enjoy :D
coin_flip
```

### Method 2 - Building directly from Github
```bash
pip install "git+https://github.com/AllyssonOFA/coin_flip.git"
```
